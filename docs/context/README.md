# Context Docs Index

This folder stores non-runtime project knowledge used for planning, content strategy, and AI-assisted editing.

## Structure

- `project/` — site architecture, taxonomy, page briefs, offers, and repo map
- `strategy/` — market/positioning analysis
- `repo/` — repository audits and technical observations

## Usage Rule

Treat everything in `docs/context/` as reference material. App behavior should be driven by source code in `app/`, `components/`, `data/`, and `lib/`.
