This file is the fast-build brief set for the highest-value pages on the site.

Each brief should help Claude or any other build agent create commercially useful pages quickly without defaulting to generic portfolio copy.

## 1. Home

- Goal: Establish immediate credibility, explain what Darling MarTech does, show proof fast, and route qualified visitors into services, work, labs, or contact.
- Target audience: Founders, operators, and growth-minded leaders with revenue pressure, messy systems, or weak digital infrastructure.
- Primary keyword/theme: Indianapolis marketing and technology partner; strategy plus systems; marketing that actually works.
- Primary CTA: Request a MarTech Audit.
- Supporting proof: Graston Technique, Pike Medical Consultants, Hoosier Boy Barbershop, 317 BBQ, CMO Simulator.
- Related services: Growth Strategy, Website & Conversion Systems, CRM / Automation / RevOps, Search Visibility, Analytics, AI Workflows.
- Related work/labs: Graston Technique, Pike Medical Consultants, 317 BBQ, CMO Simulator.
- FAQ ideas: What kind of companies are the best fit? Do you only work in Indianapolis? Are you an agency or a solo freelancer? What does a MarTech Audit include? Can you help if we already have a team?

## Rules For Claude

- Every page must sell a decision, not just describe a capability.
- Every page needs proof above the fold or immediately after the hero.
- Every page should point to a next step that matches buyer intent.
- Service pages should link to work and labs that support the specific claim on the page.
- Case studies should not be dumped generically; each example must reinforce the page's commercial argument.
